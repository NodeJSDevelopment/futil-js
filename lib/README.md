A collection of F(unctional) Util(ities). Resistance is futile.

Mostly, these are generic utilities that could conceivably be part of a library like lodash/fp.js, but for some reason or other are not.

# Docs

https://smartprocure.github.io/futil-js/

# Version History/Changelog

See our [changelog](https://github.com/smartprocure/futil-js/blob/master/CHANGELOG.md)

# Installing

`npm i -S futil` or `npm i -S futil-js`


# Usage

```js
import * as F from futil
import F from futil
import {x,y,z} from futil
```
